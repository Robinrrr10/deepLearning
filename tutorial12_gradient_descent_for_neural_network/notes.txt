Lets say if we have few data set which has X and Y with simple dataset with number, we might able to predict the formula

Lets take the dataset which has age and affordability as input data set X
And the person buy insurance or not as the output data set y
This is binnary classification which has the simple function.
For this we will be having the simple neural network which has logistic regression as the input layer and sigmoid as the output layer

Here input will be passed and then we will get the output. This is forward path

First the input will be passed to the neural network, and this will give the predicted value and we will have the actual value.
And then the error will be calculated with the formula.
This will be repeated for each of the sample data. until last data set.
After last data set, the total error will be summed and divided by number of errors to calculate the error average
Here we are using log error or cross entropy to calculate the error. For gradient descent this is best fit error rate
The value of total error will be given as log loss once it gone through all sample
Going through data set 1 time completely is the 1 epoch.
Our goal is to reduce the log loss.
So the value will be back propagated to train again with multiple epoch (multiple cycle of whole run) until the log loss was reducing.
This is gradient descent.
To get best result, it will be run with many epoch.

Here it is trying to find the derivatives where for the given change input what is the change in output.
Below is the formula

w1 = w1 - somthing
w1 = w1 - learning rate * a/aw1
w2 = w2 - learning rate * a/aw2
b = b - learning rate * a/ab

//Learning rate is the how much it changes for each input

a/aw1 = 1/n nEi=1 xi(yi^ - yi)
a/b = 1/n nEi=1 (yi^ - yi)

We will get w1, w2 and b using this formula at the end of first epoch
Then this value will be used in the formula in the forward path and then it will get the error for each and get w1, w2 and b at the end of second epoch
Similar way it will be for next and other epoch
This is batch gradient. There are other types of gradient technique as well
epoch is one of parameter which will be used in model.fit training method

At what point we need to stop the epoch or how many epoch we need to give
At one point of time the error will be very less and there wont be any change in loss.
The derivatives remail same. During that time we need to stop or we need to give that many number of epoch
This minimum value of derivatives is called global miima
This is the way gradient descent works.

def sigmoid(x):
    import math
    return 1 / (1 + math.exp(-x))

def predicted_function(age, affordability):
    weighted_sum = coef[0] * age + coef[1] * affordability + intercept
    return sigmoid(weighted_sum)

def log_loss(y_true, y_predicted):
    epsilon = 1e-15
    y_predicted_new = [max(i, epsilon) for i in y_predicted]
    y_predicted_new = [min(i, 1-epsilon) for i in y_predicted_new]
    y_predicted_new = np.array(y_predicted_new)
    return -np.mean(y_true*np.log(y_predicted_new) + (1 - y_true) * np.log(1 - y_predicted_new))

def sigmoid_numpy(X):
    return 1 / (1 + np.exp(-X))

def gradient_descent(age, affordability, y_true, epochs, loss_threshold):
    # we need to find w1, w2 and bias
    w1 = w2 = 1
    bias = 0
    rate = 0.5
    n = len(age)

    for i in range(epochs):
        weighted_sum = w1 * age + w2 * affordability + bias
        y_predicted = sigmoid_numpy(weighted_sum)
        loss = log_loss(y_true, y_predicted)
        w1d = (1/n)*np.dot(np.transpose(age), (y_predicted - y_true))
        w2d = (1/n)*np.dot(np.transpose(affordability), (y_predicted - y_true))
        bias_d = np.mean(y_predicted - y_true)
        w1 = w1 - rate * w1d
        w2 = w2 - rate * w2d
        bias = bias - rate * bias_d
        print(f'epoch:{i}, w1:{w1}, w2:{w2}, loss:{loss}')
        if loss <=loss_threshold:
            break
    return w1, w2, bias
